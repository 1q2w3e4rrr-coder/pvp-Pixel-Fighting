# pvp-Pixel-Fighting

BVN 像素格斗项目。  
本项目使用 Python 和 Pygame 实现，目标是尽量复刻经典横版像素 PVP 格斗游戏的角色选择、战斗场景、人物动作、技能特效和碰撞判定。

## 项目简介

这个项目目前已经完成了基础可玩的框架，包括：

- 主菜单
- 角色选择界面
- 战斗场景切换
- 两名角色进入战斗
- 基础血条系统
- 普攻、跳跃、受击、技能、特效、命中判定
- 战斗前加载场景和部分资源缓存

当前已接入的角色包括：

- Pain
- Naruto
- Hashirama

## 项目结构

```text
assets/    游戏素材资源
core/      核心逻辑，例如动画、状态、场景、资源加载
players/   角色类与角色动作、技能逻辑
ui/        菜单、选人界面、血条、加载界面等
main.py    程序入口
