BVN Aizen Step25 HOTFIX - Effect Height + Original Dash Distance
============================================================

User issue:
1) All skill / attack effects were visibly above the character.
2) L dodge / shunpo travelled too far and kept sliding.

Fix A - effect vertical registration:
- Original SWF FighterMain.x/y is the fighter combat/effect registration point.
- Current Godot ManifestFighter Node2D.position is the exported PNG texture center.
- Aizen idle PNG transparent bbox shows visible body below texture center, and previous code already stored ORIGINAL_FIGHTER_FOOT_OFFSET_Y.
- Therefore this hotfix converts Godot texture-center position -> original SWF fighter registration for visual effects:
  effect_y = node.position.y + ORIGINAL_FIGHTER_FOOT_OFFSET_Y

Applied to:
- bsmc followTarget attacker: currentTarget.y - 25, after converting currentTarget to SWF fighter registration.
- zh3mc attacker: caster fighter registration y at creation time.
- bisha XG_bs / XG_cbs: caster.y - 50 after converting caster to SWF fighter registration.
- dash / dash_air common effect: fighter registration y.
- hit / defense / steel / break / catch common effects: visual-only y correction; collision rectangles are not changed.
- KO hit_end effect: loser fighter registration y.

Original source references used:
- Aizen mc_023.xml:
  bsmc addAttacker("bsmc", {x:{followTarget:true,offset:0}, y:{followTarget:true,offset:-25}, applyG:false})
  zh3mc addAttacker("zh3mc", {applyG:false})
- EffectCtrler.bisha:
  doEffectById(effectId, target.x, target.y - 50, target.direct)

Fix B - L dodge / shunpo:
- Removed manual instant displacement: p1.position.x += 180 * facing.
- Original Aizen mc_023.xml shunpo timeline:
  frame 28: parent.$mc_ctrler.dash(4)
  frame 33: parent.$mc_ctrler.dashStop()
- Original FighterMain.speed = 6 px/frame.
- Godot velocity now converts px/frame -> px/s using 60 render fps.
- dashStop damping now converts original per-frame damping to px/s^2, removing the old long slide.

No changes:
- No generated images.
- No HUD / map / QiBar / HITS / texture-quality changes.
- No collision boxes changed except bsmc/zh3 attacker origin follows corrected original SWF registration.
